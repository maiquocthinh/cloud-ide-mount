# Theo Dõi Task Theo Phase

Hub này liệt kê các phase đang active và link đến file task chi tiết.

---

## Đang Active

- Phase 0: Thiết lập → [tasks/phase-0-setup.md](./tasks/phase-0-setup.md)
- Phase 1: Critical — v0.1 → [tasks/phase-1-critical.md](./tasks/phase-1-critical.md)

## Chờ

- Phase 2: High — v0.2 → [tasks/phase-2-high.md](./tasks/phase-2-high.md)
- Phase 3: Production — v1.0 → [tasks/phase-3-production.md](./tasks/phase-3-production.md)

---

## Template File Task

Khi tạo file task mới trong `tasks/`, dùng format sau:

```markdown
# Tên Phase

## Mục Tiêu
<!-- Phase này giải quyết vấn đề gì? -->

## Danh Sách Task

### Hạng mục A
- [ ] Task 1: Mô tả ngắn
  - File: `đường/dẫn/file.go`
  - Ghi chú: ...
- [ ] Task 2: ...

### Hạng mục B
- [ ] Task 1: ...
- [ ] Task 2: ...

## Ghi Chú
<!-- Quyết định thiết kế, lưu ý, v.v. -->
```
